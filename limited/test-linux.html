Installing Together Test on Linux

Welcome to the Linux install page for Together Test. Here is the link to the tarfile that will install Together Test on your Linux computer: together-test.tar.gz

Together supports Linux, but still needs testing. Echo Cancellation does not currently work on Linux. Until it is there, you must wear headphones so that others do not hear their echo when your microphone is on. If you do not have headphones, then you can use the Push to talk feature.

Instructions

Download the tarfile file linked at the top of this page. This file is small and will download quickly. Install it as you would any tarfile. Some key things you need to know are:

  • The Together launcher starts from the green icon and shows a desert oasis.
  • The launcher will always update Together when it starts.
  • While Together is running, it shows as a blue icon on your dock.
